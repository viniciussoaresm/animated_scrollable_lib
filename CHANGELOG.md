# CHANGELOG

## 1.0.4
* **Documentação**: Adicionados comentários dartdoc para todos os elementos públicos da API.
* **Pontuação**: Melhoria na pontuação de análise do pub.dev através da documentação de `AnimatedScrollableScanfold`, `AnimatedHeaderDelegate` e `HeaderBuilder`.
* **Linter**: Ativada a regra `public_member_api_docs` para garantir clareza em futuras atualizações.
* **Manutenção**: Pequenos ajustes internos nos comentários para facilitar a manutenção do código.

## 1.0.3
* **Testes Unitários:** Implementação de suíte de testes de widget para garantir integridade do scroll e cálculos de percentual.
* **Coverage:** Adicionado suporte para relatório de cobertura de código.
* **Documentação:** README atualizado com instruções de teste e badges.

## 1.0.2
* **Exemplo de Uso:** Adicionada pasta `example/` com um projeto funcional para demonstração no Pub.dev.
* **Correção de Layout:** Ajuste no `OverflowBox` para garantir que o header permaneça fixo no topo (`Alignment.topCenter`).

## 1.0.1
* **SEO e Localização:** Descrição e documentação traduzidas para Português.
* **Tags de Busca:** Adição de tópicos (`topics`) no pubspec para melhor indexação.

## 1.0.0
* **Lançamento Inicial:** Estrutura base do AnimatedScrollableScanfold com suporte a header dinâmico e rodapé fixo.
